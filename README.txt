Wild MiniGolf Challenge

Gameplay:
In Wild Minigolf Challenge, pick your name and compete against AI golfers,
complete a series of 9 different golf holes to take a spot on the leaderboards.
Completing each levels provides the player with a new customization option.
Be careful, scoring higher than the Par requires you to restart the level and lose
a life! Losing all lives causes the game to end.

Controls:
Charge -> Hold left mouse button
Release Charge -> Release left mouse button
Cancel Charge -> E
Open Settings Mid Game -> ESC

How to open the game:
Open the Zip file
Run the exectuable file
Type in a name
Play the game!
